Le Four Intelligent

Site web - Python/HTML/CSS/JS/PHP - Travail de groupe - Scolaire

Le Four Intelligent est un projet d'un an réalisé pendant ma dernière année de lycée. L'objectif était de créer un site web pour les personnes malvoyantes ayant du mal à cuisiner les plats pre-préparés puisque, les conseils d'utilisation sont souvent écrits avec une police trop petite. Le site web les aide dans cette tâche qutotidienne en configurant leur four pour l'aliment sauvegardé au préalable sur leur compte. La gestion des informations des utilisateurs inscrits se fait par le biais de progammes en python ainsi qu'en php avec les informations les plus sensibles cryptées.

Le Jeu du Serpent

Jeux Vidéo - C89 - Travail de groupe - Scolaire

Le Jeu du Serpent est un de mes premiers projets réalisé pendant mon DUT Informatique. Les instructions sont très simples : le joueur contrôle un serpent constamment en mouvement qui ramasse des pastilles afin de gagner des points et, par la même occasion, grandir. Le projet doit comporter un jeu du serpent avec quelques règles en plus notamment un chronomètre, un bouton pour mettre en pause le jeu ou pour le quitter ainsi qu'un système de score. Pour l'aspect graphique du jeu, l'université a mis en place une bibliothèque graphique.
